# Premium Reservation System

이 프로젝트는 깔끔하고 세련된 디자인의 프리미엄 예약 시스템입니다.

## 주요 기능
- **세련된 디자인**: Glassmorphism 디자인과 부드러운 애니메이션 적용
- **지정된 시간대**: 06:00, 09:00, 13:00, 16:00, 19:00 간격으로 예약 가능
- **이름 입력**: 예약 시 모달 창을 통해 이름을 입력받음
- **동시 예약 방지**: 이미 예약된 시간대는 선택할 수 없도록 구성
- **날짜 선택**: 현재부터 7일간의 날짜를 선택하여 예약 가능
- **데이터 보존**: LocalStorage를 사용하여 브라우저를 새로고침해도 예약 정보 유지

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 기술 스택
- React
